# 🚗 Vehicle Service Management System

A Spring Boot web application to manage vehicle service bookings.

## Features
- Add new booking
- Update booking
- Delete booking
- Pagination
- Search by vehicle number
- Filter by booking status

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Thymeleaf
- Bootstrap

## Run the Project

1. Clone the repository

git clone https://github.com/sairamtarra/vehicle-service-management.git

2. Configure MySQL in `application.yaml`

3. Run

mvn spring-boot:run
